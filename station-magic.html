STATION MAGIC

Ambiente de Jogos focado em Web Design (Projeto Senai)

ANO: 2023 PLATAFORMA: Web / Browser

Sobre o Projeto

O Station Magic foi um dos meus primeiros grandes desafios de front-end, desenvolvido durante o curso de Técnico em Desenvolvimento de Sistemas no SENAI Votuporanga, SP. A ideia conceitual era arquitetar uma plataforma de jogos imersiva, com paleta de cores envolventes, aplicando fundamentalmente HTML estruturado e folhas de estilo refinadas.

Por tratar-se de um projeto acadêmico de descoberta, meu objetivo principal foi o mapeamento de classes, posicionamento de FlexBox e CSS Grid, além da criação de um layout consistente e divertido para o usuário voltado à temática interativa.

Tecnologias Utilizadas

HTML5 (Semântico) CSS3 (Vanilla Animations) JavaScript Figma (UI/UX)

Destaques Técnicos

  • Navegação Guiada: Construção de menus pegajosos (sticky) e fluxos horizontais para os carrosséis de jogo.
  • Foco em Design UI: Utilização da teoria das cores com tons verdes luminosos (`#10b981`) de alta conversão adaptados à usabilidade web.
  • Organização Colaborativa: Documentação e versionamento no repositório final como um squad front-end no laboratório do Senai.
  • Hospedagem PaaS: Deploy contínuo da aplicação web usando o modelo estático via Netlify.
