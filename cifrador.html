CIFRADOR

Ferramenta Inteligente para Automação de Cifras Musicais

ANO: 2025 PLATAFORMA: Web (Next.js 16)

Sobre o Projeto

O Cifrador é uma ferramenta web que transforma nomes de músicas ou links do YouTube em documentos formatados (DOCX/PDF) prontos para uso. Ele conecta-se diretamente à API do CifraClub para buscar cifras, processa o conteúdo com transposição de acordes, identificação de seções e gera documentos profissionalmente formatados.

A ideia surgiu da necessidade de músicos que desejam ter suas cifras personalizadas, com cores customizadas por seção, transposição de tom e opção de ocultar tablaturas — tudo exportável em formato editável (DOCX) ou estático (PDF).

Tecnologias Utilizadas

Next.js 16 React 19 TypeScript Tailwind CSS 4 Framer Motion Cheerio docx pdf-lib Lucide React Vercel

Funcionalidades Principais

  • Busca Inteligente: Pesquisa diretamente na API do CifraClub. Suporta nome da música, artista ou link direto.
  • Link do YouTube: Cole um link do YouTube e o Cifrador extrai automaticamente o título e busca a cifra correspondente.
  • Transposição em Tempo Real: Transponha o tom da cifra de -12 a +12 semitons com preservação de espaçamento.
  • Cores Customizáveis: Personalize as cores de cada seção — Títulos, Acordes, Letras, Refrão, Pré-Refrão e Ponte.
  • Layouts Adaptáveis: Formato A4 (Desktop) ou Mobile (10×30cm) para leitura em dispositivos móveis.
  • Ocultar Tablaturas: Remove tablaturas diretamente do DOM + limpeza algorítmica.
  • Versão Simplificada: Acordes básicos do CifraClub para iniciantes.
  • Dimensionamento Inteligente: Cálculo dinâmico de tamanho de fonte para evitar quebra de linhas.
  • Multi-Formato: Exportação simultânea para DOCX (editável) e PDF (estático).
  • Seções Destacadas: Marcadores como [Refrão], [Ponte], [Pré-Refrão] recebem destaque amarelo automático.

Arquitetura

O projeto utiliza Next.js Server Actions para processar as requisições server-side: busca na API Solr do CifraClub com fallback para Google, extração com Cheerio, processamento de transposição e geração de documentos DOCX e PDF com formatação e cores personalizadas.
