CIFRAS MOBILE

App Nativo para Músicos — Transposição, Modo Performance & Design Premium

ANO: 2025 PLATAFORMA: Mobile (Android / iOS)

Sobre o Projeto

O Cifras é a versão mobile nativa do Cifrador, concebida para músicos exigentes. Desenvolvida para elevar a experiência visual e a rapidez nos palcos e ensaios, ela evita o excesso de anúncios de aplicativos comerciais trazendo uma visualização em Dark Mode minimalista, elegante e incrivelmente rápida.

Inspirado na plataforma web original, este app leva a experiência de leitura de cifras para outro nível com animações sedosas via Reanimated, modo performance em tela cheia e controle preciso de tipografia para diferentes distâncias de leitura.

Tecnologias Utilizadas

React Native Expo SDK Expo Router NativeWind (TailwindCSS) Reanimated Lucide React Native TypeScript tRPC Drizzle ORM Express

Funcionalidades Principais

  • Mecanismo de Transposição Real: Suba ou desça o tom dinamicamente mantendo os acordes milimetricamente alinhados com suas respectivas sílabas.
  • Modo Performance (Focus): Acione o foco em tela cheia na hora de tocar, sumindo com barrinhas superiores e mantendo apenas uma interface limpa.
  • Notas de Rodapé Livres: Insira anotações privadas, ritmos e ideias anexas abaixo de qualquer linha ou estrofe de uma música.
  • Pular Tabs: Uma configuração em 1 clique para limpar e ocultar todas as tablaturas complexas da tela quando você só precisa ver os acordes crus.
  • Controle Preciso de Fonte: Aumente e diminua a tipografia para leituras em diferentes distâncias.
  • Setup de Setlist & Histórico: Suas músicas curtidas (favoritos) ganham hospedagem em um histórico offline via armazenamento assíncrono interno.
  • Design Luxuoso: Transições aprimoradas via Reanimated trazendo aquele aspecto sedoso característico de apps iOS Premium.

Arquitetura

O projeto possui um backend próprio com Express + tRPC para comunicação com o app mobile, banco de dados MySQL gerenciado via Drizzle ORM, e compartilha lógica de processamento de cifras com a versão web (Cheerio, docx, pdf-lib).
