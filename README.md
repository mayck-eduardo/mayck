# Mayck Eduardo - Portfolio

Portfólio pessoal com coleção de projetos e ferramentas web.

## 🚀 Tecnologias

- **Vite** + **TypeScript**
- **Firebase** (Firestore, Authentication)
- **Vanilla CSS** + **GSAP** + **Lenis**

## 📦 Projetos

- **Cifrador** - Cifra/decifra textos
- **DayTracker** - Acompanhamento de dias
- **Estoque** - Controle de estoque
- **Evofit** - Página de academia
- **Station Magic** - Estação mágica
- **Contador** - Contador regressivo
- **Banda** - Página de banda
- **Admin** - Painel administrativo

## 🛠️ Como Rodar

```bash
npm install
npm run dev     # dev
npm run build   # produção
```
