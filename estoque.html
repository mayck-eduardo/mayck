ESTOQUE ZEN

Sistema de Gerenciamento de Estoque Completo

ANO: 2024 PLATAFORMA: Web (Backend/Fullstack)

Sobre o Projeto

O EstoqueZen é um sistema web monólito contruído sob a arquitetura clássica LAMP (com PHP puro e MySQL para persistência). A principal meta do projeto foi construir uma solução local capaz de suportar controle rígido de entradas e saídas (baixas) de produtos, cálculo em tempo real de quantidade total e interface simples para administração de uma loja física.

Tecnologias Utilizadas

PHP MySQL (Relacional) HTML5 CSS3 Apache

Funcionalidades Principais

  • Sistema de Login/Sessão: Roteamento protegido onde somente usuários autenticados via sessão do PHP podem acessar a base do estoque.
  • CRUD de Produtos: Fluxo para adicionar novos SKUs, editar preços de custo e descrição, além da deleção soft ou hard-delete quando viável.
  • Lançamentos e Baixas: Modalidade segura para retirar e repor produtos afetando o caixa virtual do sistema sem quebrar os índices do banco.
  • Design Responsivo: Embora o foco seja backend, a aplicação pode ser visualizada tanto via Desktops nos balcões de loja, quanto em painéis de Tablet para conferentes de galpão.
