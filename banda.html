BANDA

App Mobile para Gestão de Banda & Ensaios

ANO: 2025 PLATAFORMA: Mobile (Android / iOS)

Sobre o Projeto

O Banda é um aplicativo mobile desenvolvido para facilitar a organização e gestão de bandas musicais. Com sistema de autenticação e navegação por abas, o app oferece uma experiência fluida para gerenciar ensaios, repertórios e a rotina do grupo.

Construído com React Native e Expo, o projeto utiliza React Navigation para roteamento entre telas, contextos para gerenciamento de estado global e SVG para elementos visuais customizados.

Tecnologias Utilizadas

React Native Expo SDK React Navigation Auth Context react-native-svg Lucide React Native safe-area-context

Funcionalidades Principais

  • Autenticação: Sistema de login com contexto de autenticação para controle de acesso.
  • Navegação por Abas: Bottom Tabs Navigation para acesso rápido às diferentes seções do app.
  • Stack Navigation: Navegação em pilha para telas de detalhe e fluxos secundários.
  • Interface Responsiva: Suporte a Safe Area Context para adaptação em diferentes dispositivos.
  • Elementos Visuais SVG: Ícones e gráficos vetoriais via react-native-svg e Lucide.
